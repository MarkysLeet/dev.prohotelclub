## Заметки по проекту 5
- Новости загружаются из таблицы `news` в Supabase (поля: id, title, content, image_url, created_at).
- Добавлена возможность отправлять комментарии по нажатию `Enter` (без Shift) во всех полях ввода (главный комментарий и ответы).
- Блок "О нас" перемещен на отдельную страницу `/about`, ссылка добавлена в бургер-меню `Header`.
- При выходе из аккаунта (логаут) исправлен баг вечной загрузки путем добавления явного редиректа на главную страницу `/` и корректировки логики в `layout.tsx`.
- Изображение отеля на странице списка (`HotelCard.tsx`) сделано полностью кликабельным (оборачивает картинку и оверлей в `Link`), кнопка "Избранное" остается функциональной благодаря правильному z-index позиционированию.
- Во время долгих загрузок отображается кастомный спиннер `LoadingSpinner` с пульсирующим текстом "Подождите немного, страница загружается...".
- Комментарии оптимизированы по размеру для лучшей читаемости, ответы скрыты под кнопкой "Показать остальные ответы", по умолчанию виден только один, подгружаются по 5 штук.
